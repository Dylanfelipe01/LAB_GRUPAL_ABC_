# Laboratorio ABC

## Descripción del proyecto

**Laboratorio ABC** es una aplicación web educativa e interactiva diseñada para facilitar el aprendizaje del abecedario en español mediante tarjetas visuales.

El proyecto presenta las **27 letras del abecedario español**, incluyendo la **Ñ**, organizadas en tarjetas interactivas. Cada tarjeta muestra una letra en su parte frontal y, al hacer clic sobre ella, se voltea para mostrar una palabra asociada y una imagen representativa.

La aplicación también permite filtrar las tarjetas para visualizar todas las letras o únicamente las vocales.

El objetivo principal del proyecto es crear una experiencia sencilla, visual e interactiva que facilite el reconocimiento de las letras y su asociación con diferentes palabras.

---

## Tecnologías utilizadas

El proyecto fue desarrollado utilizando las siguientes tecnologías:

- **HTML5**: estructura y contenido de la aplicación.
- **CSS3**: estilos, diseño visual y animaciones de las tarjetas.
- **JavaScript**: interacción, lógica de las tarjetas, filtros y contador.
- **Google Fonts**: tipografía utilizada en la interfaz.
- **Git**: control de versiones.
- **GitHub**: almacenamiento y colaboración del proyecto.
